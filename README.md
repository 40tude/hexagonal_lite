# hexa_lite

> **Warning (Linux/macOS users):** The `.cargo/` folder contains Windows-specific configuration (custom `target-dir` for OneDrive, CPU flags). Delete or rename before building:
> ```bash
> rm -rf .cargo   # or: mv .cargo .cargo.bak
> ```

## Description

* This a step by step implementation solid hexagonal architecture.
* You may want to read this set of post : https://www.40tude.fr/docs/06_programmation/rust/022_solid/solid_00.html
* Hexagonal Architecture is discussed in Episode 04


## Usage
Code are in `examples/`

```bash
cargo run --example ex00
```
